#### 在现有目录中初始化仓库
```
git init
git add *
git add LICENSE
git commit -m 'initial project version'
```
#### 克隆现有仓库
```
git clone https://github.com/libgit2/libgit2 [自定义仓库名]
```

```
已跟踪的文件有3种状态:未修改,已修改,已放入暂存区.
Unmodified, Modified, Staged
```
